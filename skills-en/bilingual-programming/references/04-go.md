# Go Bilingual Guide

## EN: Overview

Go uses godoc style. Comments above declarations, `//` for inline.

## CN: 概述

Go 使用 godoc 风格。注释在声明上方，`//` 用于行内。

---

## Function

```go
// Max returns the maximum of two values.
//
// EN: Returns a if a > b, otherwise returns b.
// CN: 如果 a > b 返回 a，否则返回 b。
//
// Example:
//
//	result := Max(1, 2)  // result == 2
func Max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
```

---

## Struct

```go
// Config holds generic configuration settings.
//
// EN: Contains key-value pairs for configuration.
// CN: 包含配置用的键值对。
//
// ThreadSafety:
//         EN: Read-only after init.
//         CN: 初始化后只读。
type Config struct {
    // EN: Configuration values.
    // CN: 配置值。
    Values map[string]interface{}
}
```

---

## Interface

```go
// Validator defines input validation interface.
//
// EN: Implement to add custom validation logic.
// CN: 实现此接口添加自定义验证逻辑。
type Validator interface {
    // EN: Checks if the input is valid.
    // CN: 检查输入是否有效。
    //
    // EN: Returns error if invalid.
    // CN: 无效则返回错误。
    Validate(input interface{}) error
}
```

---

## Inline

```go
// EN: Guard against nil map.
// CN: 防御 nil 映射。
if m == nil {
    m = make(map[string]int)
}

// EN: Check error early.
// CN: 尽早检查错误。
if err != nil {
    return nil, err
}
```

## Naming

| Type | Style | Example |
|------|-------|---------|
| Function | PascalCase | `Max` |
| Type/Struct | PascalCase | `Config` |
| Variable | camelCase | `maxValue` |
| Constant | PascalCase | `MaxRetries` |
| Error | prefixed Err | `ErrInvalidInput` |
