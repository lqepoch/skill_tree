# 文档教程技能

此技能有助于创建包含逐步说明、Mermaid图表和有组织文件结构的详细文档教程。

## 目的

创建全面的文档教程，引导用户通过以下方式完成特定流程：
- 清晰的逐步说明
- 复杂工作流程的Mermaid图表
- 有适当关注点分离的有组织文件结构
- 与各种项目类型的兼容性
- 大型文档集的可扩展格式

## 使用方法

当您需要创建详细的教程/文档时，此技能将：
1. 帮助您将流程分解为清晰的步骤
2. 生成用于工作流程可视化的Mermaid图表
3. 适当地将内容组织在SKILL.md、scripts/、references/和assets/目录中
4. 提供一致文档格式的模板

## 文件结构

```
doc-tutorial/
├── SKILL.md          # 主要技能文档
├── scripts/          # 教程生成的实用脚本
├── references/       # 参考材料和模板
└── assets/           # 图像、图表和其他媒体
```

## 核心原则

1. **原子步骤**: 每个教程步骤应专注于单一操作
2. **视觉辅助**: 使用Mermaid图表来说明复杂流程
3. **关注点分离**: 将教学内容与代码/资产分开
4. **可扩展性**: 设计适用于小型和大型文档集的模板
5. **可移植性**: 确保教程可以适应不同的项目上下文

## 创建教程

使用此技能创建新教程：

1. 确定要记录的流程/文档
2. 将其分解为离散的、可操作的步骤
3. 对于每个步骤，指定：
   - 用户应该做什么
   - 他们应该在哪里点击/导航
   - 他们应该创建/修改什么
   - 预期结果
4. 创建Mermaid图表用于工作流程可视化
5. 将支持文件组织到适当的目录中
6. 确保与目标项目类型的兼容性

## Mermaid图表指南

创建Mermaid图表时：
- 使用流程图表示顺序流程
- 使用序列图表示组件间的交互
- 使用状态图表示复杂的状态转换
- 保持图表简单且专注
- 清晰标记所有元素
- 使用一致的样式

## 模板文件

此技能在references/目录中包含模板文件，以帮助标准化教程创建：
- 逐步说明模板
- Mermaid图表模板
- 文件组织指南
- 跨项目兼容性说明

## 维护

为了维护使用此技能创建的教程：
- 定期审查步骤的准确性
- 当流程发生变化时更新图表
- 验证所有文件路径和引用
- 确保与不断发展的项目结构兼容
- 保持语言清晰简洁
